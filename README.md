# Sparse-wsi-vit

This project makes use of [uv](https://docs.astral.sh/uv/getting-started/installation/).

To run the tests, run
```bash
uv run pytest
```

To run the experiments (doesn't work right now!), run
```bash
uv run experiments/run.py --config src/sparse_wsi_vit/configs/example_classification.py
```
